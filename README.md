# MHE

<b> Subset sum problem </b> (Problem sumy podzbioru)
</br>
https://en.wikipedia.org/wiki/Subset_sum_problem

Problem: W danym zbiorze liczb rzeczywistych znaleźć taki podzbiór liczb, który będzie sumował do danej wartości sumy. 
</br>
Dziedzina: liczby rzeczywiste
</br>
Wejście: zbiór liczb rzeczywistych, wartość sumy 
</br>
Wyjście: niepusty podzbiór sumujący się do wartości sumy
